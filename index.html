Requirements: Chrome / Edge / Opera on desktop  ·  USB data cable (not charge-only)

Step 1 — Driver: Most CYDs use a CP210x USB chip. If your CYD doesn't appear in the port list, install the driver: silabs.com CP210x driver  |  CH340 driver

Step 2 — Connect: Click CONNECT USB — select any port that appears (CP210x / USB Serial).

Step 3 — If sync fails: Use the FORCE BOOTLOADER button below, or manually: hold BOOT → press RST → release RST → release BOOT → retry connect.

🔧 Connection Troubleshooter

Port not appearing in picker?
• Use a data USB cable (many cables are charge-only — no data pins)
• Install CP210x or CH340 driver (links above)
• Try a different USB port on your PC
• On Windows: check Device Manager for "Unknown Device" or yellow ⚠

Port appears but sync fails?
• Another app (Arduino IDE, PuTTY, Flipper app) has the port open — close it
• Click FORCE BOOTLOADER — it sends the DTR/RTS magic sequence
• Or manually: hold BOOT pin → tap RST → release RST → release BOOT
• Try 9600 baud in the BAUD RATE selector if 115200 fails

Keeps timing out?
• Some CYD clones need 921600 baud — try the selector below
• Unplug, replug, then connect immediately before CYD boots into app firmware
